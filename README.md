# 🎾 X3

Sistema web desarrollado para la gestión integral de complejos de pádel.

Permite administrar empresas, canchas, reservas, usuarios y turnos desde una única plataforma, incorporando un calendario interactivo y distintos niveles de permisos según el tipo de usuario.

---

## Características

- Gestión de empresas
- Gestión de usuarios
- Gestión de canchas
- Calendario de reservas
- Dashboard con indicadores
- Gestión de pagos
- Control de roles
- Planes con límites configurables
- Autenticación segura
- Diseño responsive

---

## Tecnologías utilizadas

- JavaScript

- FullCalendar
- HTML5
